Supported file formats

Only Chatty chat logs and space delimited csv files whose columns are timestamp, username, and content are supported. Files should be utf-8 encoded to accuracy (the file will be converted to utf-8 anyway but you may lose some characters).

Downloading chat transcripts from a twitch vod directly

A small python wrapper is used (github page) to pull a vod's chat log through an undocumented part of the public twitch api. You will find that especially for long 5+ hour vods this will take a long time. We will ?? with a link or something after you have input the vod url when the processing is done. The reason it can take so long is because of a) max response size of a request will be around ~50 chat lines b) rate limiting queries to not be rude. Certainly a) is the biggest factor.
